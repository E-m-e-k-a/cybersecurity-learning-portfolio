# 🔐 Public Key Cryptography Basics

## 📘 Overview
Completed hands-on study of fundamental public key cryptography concepts and their real-world security applications in both defensive (SOC) and offensive security contexts.

---

## 🧠 Topics Covered

- **RSA Encryption**
  - Key generation using prime numbers (p, q)
  - Modulus creation (n)
  - Public and private key pair calculation

- **Diffie-Hellman Key Exchange**
  - Secure shared secret establishment
  - Discrete logarithm problem
  - Modular exponentiation

- **SSH Key Authentication**
  - RSA, ECDSA, Ed25519 key pairs
  - Passwordless secure remote access
  - Identifying weak SSH configurations

- **Digital Signatures**
  - Message hashing
  - Private key encryption of hashes
  - Integrity and authenticity verification

- **Public Key Infrastructure (PKI)**
  - Certificate Authorities (CAs)
  - Trust chains
  - TLS/HTTPS certificate validation

- **OpenPGP / GPG**
  - Email encryption
  - File signing and verification

---

## 🔎 Practical Security Applications

- Analyzing TLS handshakes during network traffic investigations
- Detecting Man-in-the-Middle (MITM) attacks through certificate anomalies
- Identifying weak SSH configurations vulnerable to brute-force attacks
- Verifying code-signing certificates in malware analysis
- Assessing cryptographic misconfigurations in real-world environments
